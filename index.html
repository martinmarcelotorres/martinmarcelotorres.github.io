DESARROLLADOR DEL SISTEMA

Sobre el Sistema

DELIPIZZA ES UN SISTEMA QUE TIENE COMO PRINCIPAL OBJETIVO GESTIONAR LAS VENTAS REALIZADAS DENTRO DE UNA PIZZERIA BRINDANDO AL CLIENTE UNA ATENCION AMENA, PERSONALIZADA Y RAPIDA

SU PRINCIPAL CARACTERÍSTICA RADICA EN EL CONTROL TOTAL DE NUESTROS CLIENTES, PRODUCTOS Y VENTAS REALIZADAS

DESCRIPCION DEL SISTEMA

  • OBJETIVOS ESPECIFICOS

    - INSPECCION Y CONTROL DEL STOCK DE LOS PRODUCTOS
    - DETALLAR Y ORGANIZAR LAS VENTAS REALIZADAS
    - REGISTRO, LISTA Y DETALLE DE LOS CLIENTES (PAGINA WEB O PRESENCIAL)

  • ALCANCES DEL SISTEMA

    - REGISTRO DE PERSONAS (CLIENTES Y EMPLEADOS).
    - REPORTES Y LISTADO DE PERSONAS SEGUN SU TIPO.
    - REPORTE DE PRODUCTOS DISPONIBLES Y MENU DEL DIA
    - LISTADO DE PRODUCTOS (STOCK 0 - ACTIVOS - INACTIVOS)
    - REPORTE Y DETALLE DE VENTAS
    - REGISTRO Y LISTADO DE VENTAS

  • REQUERIMIENTOS FUNCIONALES

    - EL SISTEMA ENVIARA MENSAJES DE VALIDACION, CONFIRMACION Y ERROR.
    - EL SISTEMA CUENTA CON UN FORMULARIO DE REGISTRO DE PERSONAS (ADMINISTRADOR).
    - EL SISTEMA POSEE VALIDACIONES Y API's
    - EL SISTEMA PERMITE LA ACTUALIZACION Y ELIMINACIONDE DATOS DE USUARIO (ADMINISTRADOR).
    - EL SISTEMA PERMITE CONTROLAR E INSPECCIONAR LOS PRODUCTOS (STOCK - MENU DEL DIA)
    - EL SISTEMA PERMITE EMITIR COMPROBANTES DE VENTA Y DETALLE DE LAS VENTAS
    - EL SISTEMA PERMITE REGISTRAR LAS VENTAS
    - EL SISTEMA CUENTA CON UN FILTRO DE ACCESO SEGUN TIPO DE PERSONA

  • REQUERIMIENTOS NO FUNCIONALES

    - DISEÑO DEL SITEMA (RESPONSIVE).
    - DISPONIBILIDAD DEL SISTEMA PARA LOS EMPLEADOS Y ADMINISTRADORES
    - EL TIPO DE IDIOMA DEL SISTEMA
    - SOLO EL ADMINITRADOR PODRA REGISTRAR EMPLEADOS
    - MANUAL DE USO PARA (ADMINISTRADOR Y CAJERO)
